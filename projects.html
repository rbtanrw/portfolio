DrinkMore

A web application to keep people hydrated, integrated with IoT for real-time temperature monitoring and AI for drinking action detection.


Main features of DrinkMore are:

  • Real-time temperature detection
  • Drinking reminder schedule based on current temperature
  • Notification for drinking reminder
  • Drinking action detection which will automatically turn off the ongoing notification

In this project, I contributed as an AI Engineer and Website Developer.

PesenPilem

A full stack web application for online movie ticket ordering. The front-end was built using React JS, while the back-end was built using ASP.NET Core. For the database, it uses SQL Server.


PesenPilem has the functions to:

  • Login and Sign Up
  • See "Now Showing" Movies
  • Book Seats
  • See Transaction History and Details
  • (For ADMIN) Post and Delete Movies

In this project, I contributed fully as a back-end developer.

SleepBuddy

An online website for remote hotel room reservation. The front-end was built using HTML, CSS, JS, while the back-end was built using ASP.NET Core. For the database, it uses SQL Server.

Unfortunately, SleepBuddy is not fully developed, since its front-end and its back-end are still not connected.


In this project, I contributed fully as a back-end developer.


SleepBuddy's backend has some functions:

  • Login, Sign Up, and Edit Profile
  • Search Hotels by Name, Rating, or City
  • Check Availability of Rooms
  • Reserve Rooms
  • Check Payments History
